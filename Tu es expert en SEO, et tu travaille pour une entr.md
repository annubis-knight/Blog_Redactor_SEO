# Ajoute une propriété "theme".

La propriété thème est définit par les articles intermédiaire, et chaque article spécilisé est associé à un des thème définit par l'article de niveau 2. C'est comme si il y avait des sous cocons.

Voici la structure JSON enrichie avec la propriété **theme** pour chaque article.

- Les articles de niveau intermédiaire (niveau 2) définissent les thèmes.
- Les articles spécialisés (niveau 3) héritent du thème de l’article intermédiaire auquel ils sont rattachés.
- Les articles piliers n’ont pas de thème (valeur `null`).

```json
{
  "cocons_semantiques": [
    {
      "nom": "Refonte de site web pour PME",
      "articles": [
        {
```
